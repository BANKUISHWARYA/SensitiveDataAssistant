# Sensitive Data Detection & Compliance Assistant

An AI-powered Streamlit application that detects sensitive information in uploaded PDF, TXT, and CSV files.

## Features

- Upload PDF/TXT/CSV
- Detect PII using Regex
- Risk Classification
- AI Compliance Summary
- Gemini AI Q&A
- PDF Compliance Report

## Technologies

- Python
- Streamlit
- Pandas
- Regex
- ReportLab
- Google Gemini AI

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
